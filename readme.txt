Readme file for HW 3:

The program was created in c++ and will run in putty.

First, compile the program, type:
g++ shopping.cpp -o shopping

This creates a program named "shopping"

Next, run the file, type:
./shopping

This program takes in a text file called shopping.txt that must be in the same directory.
A file called results.txt will be created after running the program "shopping".
